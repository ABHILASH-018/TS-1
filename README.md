## Description

This project is a responsive landing page built using HTML5 and Tailwind CSS. It includes a navigation bar, a hero section with call-to-action buttons, a dashboard preview, floating status cards, and a fixed chat widget. The project demonstrates the use of semantic HTML and responsive design principles.

## How to Run

1. Download or clone the project files.
2. Ensure `index.html` is in the project folder.
3. Open `index.html` in any modern web browser (such as Chrome, Edge, or Firefox).
4. An internet connection is required because Tailwind CSS is loaded through its CDN.

## Files

* `index.html` – Main webpage.
* `README.md` – Project documentation and instructions.
